## Projects

### 🐾 [eduCATe.me](https://github.com/Vera-Kibin/eduCATe.me)

A role-based Learning Management System (LMS) built as a React Single Page Application.
The project simulates a real-world educational platform and includes:

- role-based UI (Student / Instructor / Admin)
- course structure with modules and lessons
- quiz system and progress tracking
- gamification (XP, levels, badges)
- video + transcript synchronization
- drag-and-drop course editor
- threaded discussions with rich text and sanitization

Focused on frontend architecture, state management, and complex UI logic.

### 🧪 [PurrTasks](https://github.com/Vera-Kibin/PurrTasks)

A full-stack task management system built with Flask and React.
The project focuses on clean architecture and backend design, featuring:

- role-based task management (owner / manager)
- task lifecycle and status transitions
- filtering, history tracking, and validation logic
- support for both in-memory and MongoDB storage
- comprehensive testing (unit, API, BDD, performance, UI)

Designed to simulate a production-like backend system with strong emphasis on testing and maintainability.

### 🏦 [Banking API Testing Project](https://github.com/Vera-Kibin/banking-api-testing-project)

A backend-focused project for testing financial systems.  
Includes unit, API, BDD, and performance testing.

### 🌍 [LivingWorld](https://github.com/Vera-Kibin/LivingWorld)

An interactive ecosystem simulation built in C++ with Qt.

The project focuses on object-oriented design and real-time simulation, featuring:

- tile-based world with different terrains
- dynamic organism behavior (animals and plants)
- turn-based simulation engine
- GUI built with Qt Widgets
- OOP architecture with inheritance and polymorphism

Designed to demonstrate practical use of C++ and OOP in a complex interactive system.
